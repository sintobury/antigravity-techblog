Next.js는 풀스택 웹 애플리케이션을 구축하기 위한 React 프레임워크입니다. 사용자 인터페이스를 구축하기 위해 React Component를 사용하고, 추가적인 기능과 최적화를 위해 Next.js를 사용합니다.

## 주요 기능

### Routing
Server Component 위에 구축된 파일 시스템 기반 라우터로, 레이아웃, 중첩 라우팅, 로딩 상태, 에러 핸들링 등을 지원합니다.

### Rendering
Client Component와 Server Component를 사용한 클라이언트 사이드 및 서버 사이드 렌더링을 지원합니다.

### Data Fetching
Server Component에서 async/await를 사용하여 데이터 페칭을 단순화했습니다.

## 왜 Next.js인가?

React에 필요한 툴링과 설정을 처리하며, 애플리케이션을 위한 추가적인 구조, 기능, 최적화를 제공합니다.

### Developer Experience
Next.js는 Fast Refresh와 TypeScript 기본 지원 등 뛰어난 개발자 경험을 제공합니다.

### Performance
자동 이미지 최적화, 폰트 최적화, 스크립트 최적화를 통해 애플리케이션이 기본적으로 빠르게 작동하도록 돕습니다.
